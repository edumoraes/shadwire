# Shadwire MVP Phase 1: Foundation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Bootstrap the Shadwire monorepo foundation: root project files, Rails sandbox, registry manifest, sync script, shared component base, helpers, and Tailwind v4 tokens.

**Architecture:** `registry/` is the source of truth for distributable component files. `sandbox/` is a Rails app that consumes registry files via a local sync script and runs previews/tests against copied code. `packages/cli/` is reserved so the registry schema can support a CLI in a later phase without changing the MVP shape.

**Tech Stack:** Ruby 3.4, Rails, ViewComponent, Tailwind CSS v4, Turbo, Stimulus, Minitest, Capybara assertions.

- [ ] **Step 2: Add the sync script**

Create `bin/sync_registry`:

```ruby
#!/usr/bin/env ruby
# frozen_string_literal: true

require "fileutils"
require "json"
require "pathname"

ROOT = Pathname.new(__dir__).join("..").expand_path
REGISTRY = JSON.parse(ROOT.join("registry/registry.json").read)
SANDBOX = ROOT.join("sandbox")

unless SANDBOX.directory?
  warn "sandbox/ does not exist. Run Task 2 before syncing registry files."
  exit 1
end

files = REGISTRY.fetch("items").flat_map { |item| item.fetch("files") }.uniq

files.each do |file|
  source = ROOT.join(file.fetch("source"))
  target = SANDBOX.join(file.fetch("target"))

  unless source.file?
    warn "Missing registry source: #{source.relative_path_from(ROOT)}"
    exit 1
  end

  FileUtils.mkdir_p(target.dirname)
  FileUtils.cp(source, target)
  puts "synced #{source.relative_path_from(ROOT)} -> #{target.relative_path_from(ROOT)}"
end
```

Run:

```bash

mkdir -p registry/rails/ui/javascript/controllers
```

Create `registry/rails/ui/components/ui_component.rb`:

```ruby
# frozen_string_literal: true

class UiComponent < ViewComponent::Base
  private

  def extract_class_name(attrs, class_name)
    [attrs.delete(:class), class_name].compact_blank.join(" ")
  end

  def fetch_variant(mapping, key, fallback:)
    mapping.fetch(key.to_sym) { mapping.fetch(fallback) }
  end

  def html_attrs
    @attrs
  end
end
```
